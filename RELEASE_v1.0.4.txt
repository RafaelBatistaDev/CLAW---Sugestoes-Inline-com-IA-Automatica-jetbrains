📥 INSTALLATION
═════════════════════════════════════════════════════════════════════════════════

Method 1: JetBrains Marketplace (Recommended)
  1. Open your JetBrains IDE
  2. Go to Settings → Plugins → Marketplace
  3. Search for "CLAW"
  4. Click "Install"
  5. Restart IDE

Method 2: Manual Installation
  1. Download claw-jetbrains-rider-plugin-1.0.4.zip
  2. Open IDE → Settings → Plugins → ⚙️ → "Install Plugin from Disk..."
  3. Select the ZIP file
  4. Click "OK"
  5. Restart IDE
