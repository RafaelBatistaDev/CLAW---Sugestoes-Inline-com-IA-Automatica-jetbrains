## 🎯 Installation Instructions

### Option 1: JetBrains Marketplace (Recommended)
1. Open your JetBrains IDE (IntelliJ, Rider, etc.)
2. Go to Settings → Plugins → Marketplace
3. Search for "CLAW"
4. Click "Install"
5. Restart IDE

### Option 2: Manual Installation
1. Extract or download `claw-jetbrains-rider-plugin-1.0.4.zip`
2. In IDE: Settings → Plugins → ⚙️ → "Install Plugin from Disk..."
3. Select the ZIP file
4. Click "OK"
5. Restart IDE

### Option 3: From GitHub
1. Go to GitHub Release page
2. Download the latest ZIP from Releases
3. Follow Option 2
